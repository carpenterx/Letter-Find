Instructions

  1. Input all the valid letters in the "All letters" field
  2. Select the letter that needs to be found
  3. Select how many instances of the correct letter will appear on the page
  4. Check "Allow lowercase" in order to have lowercase letters appear as well
  5. Click "Refresh" to generate a new page with the current settings
  6. Click the "Print" button to print the page or save it as a PDF

Tips

  1. You do not need to use all the letters of the alphabet in order to generate a page
  2. You can input a letter multiple times in order to make it more likely to appear on the page
  3. You can click on the letters on the page to fill the circles with color. Click the letter again to remove the fill
  4. If you wish to print the page with the fills visible, go to "Print", expand the "More settings" menu and check "Background graphics"
